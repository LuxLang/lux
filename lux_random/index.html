lux/random

Pseudo-random number generation (PRNG) algorithms.

Types

PRNG

An abstract way to represent any PRNG.

(type: #rec PRNG
  (-> lux;Unit [PRNG lux;Nat]))

Random

A producer of random values based on a PRNG.

(type: (Random a)
  (-> PRNG [PRNG a]))
